Légende

  • Navigateur Web : le client envoie des requêtes HTTP classiques (formulaires) et des requêtes AJAX (chatbot)
  • Middleware Django : 7 couches de traitement (sécurité, session, CSRF, authentification, messages)
  • 4 Apps Django : accounts (auth), chatbot (Q&A), campus (vie univ.), reporting (analytics)
  • SQLite3 : base de données relationnelle locale (db.sqlite3)
  • Power BI : connecté via exports CSV ou API JSON avec clé API
