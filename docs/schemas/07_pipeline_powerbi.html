7.2 — Méthode 1 : Exports CSV

CSV Nécessite une session admin Django active. Fichiers encodés UTF-8 BOM avec séparateur point-virgule (;) pour compatibilité Excel/Power BI français.

EndpointDescriptionColonnes principales
/reporting/export/utilisateurs/ Liste de tous les utilisateurs Username, Nom, Prénom, Email, Rôle, Formation, Date inscription, Actif
/reporting/export/emargements/ Feuilles d'émargement Etudiant, Cours, Date, Statut présence, Formation
/reporting/export/chatbot/ Logs du chatbot Utilisateur, Question, Réponse, Match, Satisfaction, Date
/reporting/export/evenements/ Événements campus Titre, Organisateur, Date, Lieu, Statut, Nombre inscrits

7.3 — Méthode 2 : API JSON

API Authentification par clé API passée en query parameter. Pas de session requise.

EndpointParamètre requisFormat retour
/reporting/api/utilisateurs/ ?api_key=campus-connect-powerbi-2026 Array JSON d'objets utilisateur
/reporting/api/emargements/ ?api_key=... Array JSON d'émargements
/reporting/api/chatbot_logs/ ?api_key=... Array JSON de logs chatbot
/reporting/api/evenements/ ?api_key=... Array JSON d'événements
/reporting/api/cours/ ?api_key=... Array JSON de cours
